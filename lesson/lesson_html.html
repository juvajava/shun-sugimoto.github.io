2. Bracketsのプラグインをインストール

コーディングを楽にするために、プラグインを入れましょう。

右上の「フォルダマーク」をクリックするとプラグインを検索できます

最低限、以下のプラグインは入れましょう。

プラグイン 概要
Emmets

HTMLとCSSのコーディング補助ツールです。

主にタグの記載がすごく楽になります。
Beautify

コードを整形してくれます。

「編集」 > 「保存時にBeatifulyをする」をチェックしてください。

Brackets File Icons

ファイルアイコンがつきます。

見やすい。

Custom Work

タブ切り替えができるようになります。
HTML Block Selector

スペース・タブを可視化します。
Highlight Multibyte Symbols
